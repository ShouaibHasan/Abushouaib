AUDIO SPECTRUM ANALYZER

With filters, LM 3914 and IC 4017 we made it.With the music ‘s amplitude(voltage) the level of led rise and fall in the spectrum analyzer. Input signal comes into each filter. Filters are at first high pass filter and then low pass filter. They only pass signals belonged to fixed frequency range and remove others. We can use transistors and discrete capacitors and resistors to make active filters or we can use OP-AMP and other passive devices. The more filters, the higher frequency resolution of the spectrum analyzer, shown as the higher quality of the circuit.VU-LED Circuit employed to display signal level behind filters. This kind of display circuit can be discrete devices or specific ICs.

Breast Cancer detection

Along with the original photos, the ground truth images (mask image) are used to train and evaluate. Three categories—normal, benign, and malignant—are used to group the photos. The model's input photos are grayscale and they are normalized with 255. The dataset is split in train-test set with 80:20 ratio. Then the data is used to train the custom CNN model.

Vain Pattern Extraction

NIR CCD Camera-Grayscale-Adaptive Histogram-Binary Image. The uniqueness and stability of the vein patterns provide a good biometric feature for identification of the person. The proposed work is related to the implementation of Infrared vein detection and matching system for person identification. A novel scheme is proposed, in which the vein image captured in an infrared illuminator is used. The proposed system is implemented using MATLAB.

Power System Data Logger with Protection and Control Mechanism

The system will measure the voltage, current, power and power factor of a system and measure the energy flow. It will upload the data to a server every minute so that the data can be stored for future processing and the power usage trend of the system can be known. It will also contain some protection mechanisms like over-voltage, under-voltage and overload protection. It will also contain control features where any load can be individually turned off from the server if needed.
